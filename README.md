# ROS_Qt_GUI_Henes
**Install qt build for ROS Melodic**
```
sudo apt-get install ros-melodic-qt-build
```
**Create a catkin workspace**
```
mkdir -p ~/catkin_ws/src
cd catkin_ws/src/
```
**Clone repository to src folder**
```
git clone https://github.com/ ...
```
**Catkin make**
```
cd ~/catkin_ws/
catkin_make
```
